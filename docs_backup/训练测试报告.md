## 建议

### 如果想立即看效果
可以降低难度,让AI更快学会:

在 `config.py` 中:
```python
MCTS_SIMULATIONS = 10  # 降低到10(原50)
MAX_MOVES = 100        # 降低到100(原200)
```

这样:
- 训练速度快5倍
- AI更容易学到东西
- 但最终棋力会弱一些
