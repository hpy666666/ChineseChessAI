# 快速开始指南

## 🚀 5分钟上手

### 第1步：安装依赖（只需一次）

**方法A：双击运行**
```
双击 install.bat
```

**方法B：命令行**
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
pip install numpy pygame
```

等待安装完成（2-5分钟）

---

### 第2步：测试系统

**方法A：使用启动器**
```
双击 start.bat
选择 [3] 测试系统
```

**方法B：命令行**
```bash
python main.py test
```

如果看到"所有测试通过"，说明系统正常！

---

### 第3步：开始训练

**方法A：使用启动器**
```
双击 start.bat
选择 [1] 开始训练
```

**方法B：命令行**
```bash
python main.py train
```

你会看到类似输出：
```
============================================================
开始训练中国象棋AI
============================================================
设备: cuda
GPU型号: NVIDIA GeForce RTX 4070 Laptop GPU
每轮自我对弈: 100局
============================================================

【第 1/100 轮训练】

▶ 阶段1: 自我对弈 (100局)...
   进度: 10/100 局 | 红胜:5 黑胜:4 和:1
   进度: 20/100 局 | 红胜:11 黑胜:8 和:1
   ...
```

**停止训练**: 按 `Ctrl+C`（模型会自动保存）

---

### 第4步：观看AI对局

训练一段时间后（建议至少30分钟），可以观看AI的表现：

**方法A：使用启动器**
```
双击 start.bat
选择 [2] 观看对局
```

**方法B：命令行**
```bash
python main.py watch
```

会弹出图形界面，显示AI下棋！

```python
MCTS_SIMULATIONS = 25

```

## 🎯 常见问题

### ❓ 提示"找不到torch"
**解决**: 运行 `install.bat` 或手动安装PyTorch

### ❓ 提示"CUDA不可用"
**解决**:
1. 确保安装的是GPU版本的PyTorch
2. 更新NVIDIA驱动
3. 重新运行 `install.bat`

### ❓ 训练很慢
**解决**:
1. 检查输出是否显示"cuda"（应该用GPU）
2. 减小 `MCTS_SIMULATIONS` 参数
3. 关闭其他占用GPU的程序

### ❓ 内存不够
**解决**:
1. 减小 `BATCH_SIZE` 参数
2. 减小 `BUFFER_SIZE` 参数
